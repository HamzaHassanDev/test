لما تخلص تعديل في اي ملف او مجموعه منهم

تدخل علي الملف اللي انت هتحمله وتفتح git bash here

وتكتب git add . 

وده بيخليك تضيف كل الملفات ل مستودع مخفي مش ف github تقدر تقول وهمي

بعدين لو حابب تكتب التعديل او version معين تكتب

git commit -m "التعديل" مثلا "version 1.1 

بعدين عشان تدخل علي الصفحه وتحدد مكان نقل الملفات لو انت مكنتش فيها اصلا تكتب

git remote add origin واللينك اللي هديهولك

بعدين عشان ترفع الملفات

git push

طبعا في طرق تانيه عشان ترفع الملف اللي عدلت عليه بس مش الملفات كلها ولكن هتعلمها واقولك عليها والموضوع بياخد وقت ف الرفع خلي بالك

ووقت ما تتعامل مع git push تكتب كل حاجه بالكيبورد او تستخدم الماوس ف موضوع ال copy/paste لأنه ctrl c او ctrl v بيعطبو البرنامج ومش شغالين
